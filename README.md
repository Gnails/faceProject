# faceProject
project 2d face image to 3d model
we use asmlibrary of @greatyao,you can get the code in cite

Require
=======
Directx9
opencv
asmlibrary

Tutorial
========
Usage: faceProject -i image_file -m model_file -h haarcascade_file
